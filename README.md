# 404AI - Двухагентная система ИИ

## Описание
Система состоит из двух ИИ-агентов:
- **Публичный агент** - размещается на GitHub Pages, принимает вопросы пользователей
- **Локальный агент** - уведомляет о новых вопросах и позволяет отвечать на них

## Режимы работы
1. **Ручной режим** - вы отвечаете на все вопросы самостоятельно
2. **Авто-режим** - агент отвечает рандомными фразами-уходками

## Структура проекта
```
404AI/
├── public-agent/          # Публичный агент для GitHub Pages
├── local-agent/           # Локальный агент с уведомлениями
└── shared/               # Общие компоненты и стили
```

## Запуск
### Публичный агент
Разместить содержимое `public-agent/` на GitHub Pages

### Локальный агент
Запустить `local-agent/index.html` в браузере
