# Звуки для уведомлений

В эту папку нужно добавить звуковые файлы:

- `notification.mp3` - звук нового сообщения
- `send.mp3` - звук отправки сообщения  
- `error.mp3` - звук ошибки

Временное решение:
Браузер будет использовать системные звуки или создавать простые beep звуки программно.

Для полноценной работы замените файлы на свои аудиофайлы.
